# 거상 아이템 거래 서비스

## 프로젝트 개요
거상 게임 유저 간 아이템 거래 지원 서비스
Spring Boot 4.0, Java 21, MySQL, JPA

## 에이전트 역할 정의

### 1. 기획 에이전트
- PRD 기반으로 기능 목록, API 명세 초안 작성
- 우선순위 및 구현 순서 결정

### 2. 데이터설계 에이전트
- ERD 설계 (세트/피스/주술/주술세트 포함)
- JPA Entity 설계 및 연관관계 정의
- 인덱스 전략 수립

### 3. 구현 에이전트
- 기획/데이터설계 결과물 기반으로 코드 작성
- Controller/Service/Repository 레이어 구현
- 비즈니스 로직 구현

### 4. 검증 에이전트
- 구현된 코드만 보고 리뷰
- 로직 오류, 누락 기능, 보안 이슈 확인
- 수정 필요 사항 목록 작성

### 5. 테스트 에이전트
- JUnit 단위 테스트 작성
- Service 레이어 중심 테스트
- 경계값/예외 케이스 포함

### 6. 배포 에이전트
- 빌드/배포 스크립트 작성
- 환경변수 및 설정 관리

## 작업 규칙
- 각 단계는 이전 단계 결과물 기반으로 진행
- 한국어 주석 필수
- 현금 거래 관련 기능 구현 금지
