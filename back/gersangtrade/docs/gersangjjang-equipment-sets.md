# 거상짱 장비 세트 데이터

크롤링 대상: `https://www.gersangjjang.com/set/` (URL 확인 필요)  
수집 목적: `EquipmentSet` + `EquipmentSetPiece` + `EquipmentItem` + `ItemStat` 적재

---

## HTML 구조

```
.g-table
  .g-row                         ← 세트 1개
    .g-cell.lv-col               ← 세트명 + 레벨 (예: "공명\n(215lv)")
    .g-cell.item-col             ← 피스 목록 (<br> 구분, "관 (방어300, 힘50...)")
    .g-cell.effect-col strong    ← 세트 효과 (<br> 구분)
    .g-cell.max-col              ← 풀세트 최대치 (참고용, DB 미저장)
```

피스 1행 형식: `{피스명} ({스탯1, 스탯2, ...})[{N개}]`  
반지는 "2개" 접미어 포함 → `EquipmentSetPiece.pieceCount = 2`

## DB 저장 규칙

- `EquipmentSet.name` = `{세트명}세트` (예: "공명세트")
- `EquipmentSet.setType` = `"GERSANG_SET"`
- `Item.name` = `{세트명}{피스명}` (예: "공명관", "백호머리띠")
- `Item.type` = `EQUIPMENT`
- `EquipmentItem.slot` = 피스명 → 슬롯 매핑 결과
- `EquipmentItem.equipmentKind` = `NORMAL`
- `EquipmentSetPiece.pieceCount` = 1 (반지만 2)
- 세트효과는 현재 DB 미저장 (로그 출력만)
