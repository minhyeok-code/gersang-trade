# 거상인 프론트엔드 디자인 명세

프로토타입 코드(`front/docs/design/`)를 기준으로 작성한 레이아웃·스타일·페이지 구성 명세.
실제 구현 시 이 문서를 참조해 일관성을 유지한다.

---

## 1. 기술 스택

| 항목 | 내용 |
|------|------|
| 프레임워크 | React 19 + TypeScript |
| 빌드 도구 | Vite |
| 스타일 | TailwindCSS v4 |
| UI 컴포넌트 | shadcn/ui (accordion, dialog, drawer, tabs 등 풀세트) |
| 라우터 | React Router v7 (`createBrowserRouter`) |
| 아이콘 | lucide-react |

### 2-2. 타이포그래피

- 기본 폰트 크기: `16px`
- 제목·가격·브랜드명: `font-serif` (세리프 계열)
- 본문: sans-serif 기본
- h1 `text-3xl font-bold`, h2 `text-xl font-semibold`, h3 `text-lg font-semibold`
- 보조 텍스트: `text-xs text-[#7A7368]`
- 카드 가격: `font-serif text-base font-bold text-[#8B6B4A]`

### 2-3. 공통 컴포넌트 패턴

**카드**
```
bg-[#FAF8F2] border border-[#DDD6CA] rounded-lg
hover: hover:border-[#8B6B4A] hover:shadow-md transition-all
```

**버튼(기본/강조)**
```
bg-[#8B6B4A] text-[#E8DCCB] px-4 py-{n} rounded
hover:bg-[#6A4F35] transition-colors
```

**버튼(보조/윤곽선)**
```
border border-[#DDD6CA] bg-transparent text-[#7A7368] rounded
hover:border-[#8B6B4A] hover:text-[#8B6B4A]
```

**입력 필드**
```
border border-[#DDD6CA] bg-white px-4 py-2.5 rounded text-sm
outline-none focus:border-[#8B6B4A] transition-colors
disabled: disabled:bg-[#F5F1E8] disabled:text-[#7A7368]
```

**탭 언더라인**
```
active:   border-b-2 border-[#8B6B4A] text-[#3E3A36] font-medium
inactive: border-b-2 border-transparent text-[#7A7368] hover:text-[#3E3A36]
```

**모달 오버레이**
```
fixed inset-0 bg-black/65 z-[500] flex items-center justify-center
내부 카드: bg-[#FAF8F2] border border-[#DDD6CA] rounded shadow-[0_24px_60px_rgba(0,0,0,0.3)]
```

## 3. 글로벌 레이아웃

### 헤더 (`Header.tsx`)
- 높이: `52px`, `sticky top-0 z-[200]`
- 배경: `#2A2724`, 하단 테두리: `2px #8B6B4A`
- **왼쪽**: 브랜드 로고 "거상인" (`font-serif text-lg font-bold text-[#E8DCCB]`) + 서버 선택 드롭다운
- **가운데**: 네비게이션 (`메인` / `거래` / `전투 계산기`)
- **오른쪽**: 알림(Bell) · 채팅(MessageCircle) 아이콘 뱃지 + "내정보" 링크
- 서버 드롭다운: `#1E1C1A` 배경, 선택된 서버 `text-[#8B6B4A] font-medium`
- 알림·채팅 패널: `w-[320px]` 우측 팝오버, `z-[300]`

### 페이지 최대 너비
- 메인·프로필: `max-w-[1400px]` / `max-w-[1200px]`
- 거래: 전체 너비 (사이드바 + flex 레이아웃)
- 덱 빌더: `max-w-[1600px]`
- 로그인: `max-w-md` (중앙 정렬)

## 4. 페이지별 명세

### 4-1. 메인 페이지 (`/`)

**목적**: 사용자 속성 덱에 맞는 가성비 루트 제안 + 관심 아이템 시세 요약

**구성 (위→아래)**

1. **속성 배너** — 그라디언트 배너, 사용자 설정 속성(`userElement`) 기준 아이콘·제목 표시
2. **가성비 루트 이미지 2장** — `grid grid-cols-2`, `aspect-video` 카드. 이미지 + 루트명 + 예상 투자금
3. **2단 그리드** (`lg:grid-cols-2`)
   - **관심목록 최근 거래**: 아이템별 최근 5건 거래가 + 7일 평균가
   - **가성비 지표**: 대상 몬스터 기준 관심 아이템 리스트, 가성비 점수(`데미지 증가율 / 가격`), 추천 텍스트

**주의**: `userElement`는 추후 로그인 사용자의 덱 속성으로 연동

---

### 4-2. 거래 페이지 (`/trade`)

**목적**: 팝니다·삽니다 등록글 목록 조회 + 거래 상세 + 시세 검색

**레이아웃**: `flex` — 좌측 사이드바(210px) + 우측 메인

**사이드바 필터 (sticky)**
- 구분: 전체 / 단품 / 세트
- 세트명 검색 (텍스트 입력)
- 구성: 전체 / 갑투 / 변 / 풀 / 풀반쌍
- 주술: 전체 / 없음 / 북두칠성 / 천추 / 천기 / 천권 / 개양 / 요광 (체크리스트)

**탭 바 (sticky, `top-[52px]`)**
- 전체 / 팝니다 / 삽니다 탭
- 우측: "시세 검색" 버튼 (드로어 오픈) + "+ 등록하기" 버튼

**목록 영역**: `grid grid-cols-2` — 팝니다 컬럼(좌) / 삽니다 컬럼(우)
- 각 컬럼 헤더: 배지 + 건수 + 정렬(최신/낮은가/높은가)
- 거래 카드:
  - 좌측 컬러 바: 팝니다 `#A8C49A/60`, 삽니다 `#9AAAC4/60`
  - 닉네임 → 구분선 → 아이템명(serif) → 가격(serif, `#8B6B4A`) → 게임닉·접속시간·등록시간

**거래 상세 모달** (카드 클릭)
- `w-[660px]`, `grid grid-cols-[170px_1fr]`
- 왼쪽 프로필 패널: 아바타(이니셜) + 닉네임 + 등급 배지 + 서버·게임닉·접속시간
- 오른쪽 거래 정보: 아이템 아이콘 + 팝/삽 배지 + 아이템명 + 가격 + 구성·주술·등록 테이블
- 하단: 신고 버튼 + 채팅하기 버튼

**시세 검색 드로어**: shadcn `Drawer` — 아이템명 + 세트 구성 선택 + 검색

---

### 4-3. 전투 계산기 (`/deck`)

**목적**: 덱(주인공 + 용병 12명) 구성 및 장비 설정 → DPS 분석

**구성 (위→아래)**

1. **주인공 선택**: 아바타 + 이름·국적·성별 + "변경" 버튼 → 모달(10개 선택지, 2열 그리드)

2. **용병 구성(12슬롯)**: `grid grid-cols-4`
   - 슬롯 0: 주인공 고정
   - 슬롯 1~11: 비어있으면 점선 테두리 + "+" 클릭 → 용병 선택 모달
   - 용병 배치 시: 이름·카테고리 + DPS 기여도 표시 + "장비 설정" 버튼

3. **장비 설정 패널** (용병 선택 후 표시, 2열)
   - 주 장비 3×3: 부적·투구·장갑·무기·갑옷·요대·반지1·신발·반지2
   - 액세서리 2×4: 기운·목걸이·무기액세서리·귀걸이·상의·각반·하의·팔찌

4. **대상 몬스터 + 전투력 분석** (`lg:grid-cols-2`)
   - 몬스터: 속성·저항 표시 + "변경" 버튼 → 선택 모달
   - 분석: 총 DPS·저항 감소·적 속성·적 저항 수치 카드

**용병 선택 모달**: `w-[700px]`, 검색 + 카테고리 필터(전체/전사/마법사/암살자/지원/힐러) + 2열 그리드

---

### 4-4. 프로필 페이지 (`/profile`)

**목적**: 내 계정 정보 관리 + 거래 내역 조회

**레이아웃**: `lg:grid-cols-3` — 좌측 프로필 카드(1) + 우측 상세(2)

**프로필 카드**
- 원형 아바타(이니셜) + 닉네임 + 등급 배지
- 신뢰도·거래횟수·가입일 표시
- "내 덱 구성하기" 버튼 → `/deck`

**프로필 정보 (수정 가능)**
- 수정/저장/취소 토글
- 필드: 닉네임·게임닉·이메일·서버·접속가능시간

**최근 거래 내역**
- 판매(초록 배지) / 구매(파랑 배지) + 아이템명·상대방·날짜 + 가격

---

### 4-5. 로그인 페이지 (`/login`)

**목적**: 로그인 / 아이디찾기 / 회원가입

**레이아웃**: 중앙 `max-w-md` 카드

**탭 전환**: 로그인 / 아이디찾기 / 회원가입 (언더라인 탭)

- **로그인**: 아이디·비밀번호 입력 + 로그인 버튼
- **아이디찾기**: 이메일 입력 + 찾기 버튼
- **회원가입**: 아이디·이메일·비밀번호·비밀번호확인

> ⚠️ **구현 시 주의**: 백엔드는 **OAuth2(Google/Naver)만 지원**하므로 아이디/비밀번호 폼은 구현하지 않는다.
> 실제 로그인 페이지는 Google/Naver 소셜 로그인 버튼만 제공한다.

## 5. z-index 계층

| 레이어 | 값 | 대상 |
|--------|-----|------|
| 헤더 | `z-[200]` | 글로벌 헤더 |
| 사이드바/탭바 | `z-[50]` ~ `z-[100]` | sticky 요소 |
| 드롭다운 | `z-[300]` | 서버·알림·채팅 팝오버 |
| 모달/오버레이 | `z-[500]` | 거래 상세, 용병·몬스터 선택 모달 |
